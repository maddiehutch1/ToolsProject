# Architecture Document
## Mental Health Companion — Agentic Chat Application

**Version:** 1.1 | **Date:** March 9, 2026

---

## System Overview

Single FastAPI server serving both the static frontend and the `/chat` API. Every request is crisis-checked before the agent runs. The agent loop may call one or more tools before streaming the final response via SSE.

```
Browser (HTML/CSS/JS)
  └── POST /chat ──► FastAPI
        ├── detect_crisis()  →  crisis SSE  →  done
        └── LangGraph StateGraph
              ├── call_model  (gpt-4o-mini)
              └── tools
                    ├── rag_search  →  FAISS (local disk)
                    ├── web_search  →  Tavily API
                    └── calculator  →  simpleeval
        └── StreamingResponse (SSE) ──► browser
```

## Request Lifecycle

1. `app.js` → `POST /chat { message, session_id }`
2. `detect_crisis(message)` — if true, stream crisis event and return; LLM never called
3. Load `session_store[session_id]`; append `HumanMessage`
4. Pass `[SystemMessage] + full_history` to `graph.astream_events(state, version="v2")`
5. LangGraph loop: `call_model` → if `tool_calls` → `tools` → back to `call_model`
6. Stream `token`, `tool_use`, `tool_done` events as they fire; emit `done` at end
7. Append `AIMessage` to `session_store[session_id]`

## LangGraph Agent

### State
```python
class AgentState(TypedDict):
    messages: Annotated[list[BaseMessage], operator.add]  # appends, never overwrites
```

### Graph Topology
```
START → call_model ──(tool_calls?)──► tools ──┐
             ▲                                 │
             └─────────────────────────────────┘
             │ (no tool_calls)
            END
```

| Node | Implementation | What it does |
|------|---------------|-------------|
| `call_model` | Custom | `llm_with_tools.invoke([SystemMessage] + state["messages"])` |
| `tools` | `ToolNode` (prebuilt) | Dispatches `tool_calls` to matching `@tool` function; appends `ToolMessage` |

Conditional edge `should_continue`: returns `"tools"` if `last_message.tool_calls` else `END`.

### Tool Binding
`llm_with_tools = ChatOpenAI(model="gpt-4o-mini", streaming=True).bind_tools([rag_search, web_search, calculator])`

Tool schemas are derived automatically from `@tool` docstrings — no manual JSON schema needed.

## RAG Pipeline

**Ingest (run once via `backend/ingest.py`):**
```
knowledge_base/*.md  →  TextLoader  →  RecursiveCharacterTextSplitter(500/50)
  →  OpenAIEmbeddings("text-embedding-3-small")  →  FAISS.save_local("faiss_index")
```

**Retrieval (per tool call):**
```
query  →  retriever.invoke(query)  →  top-4 Documents
  →  format: "[source_filename]\n{page_content}" joined by "---"  →  ToolMessage
```

The `[source_filename]` prefix lets the LLM cite the document in its response.

**Knowledge base documents:**

| File | Content |
|------|---------|
| `cbt_techniques.md` | Thought records, cognitive restructuring, behavioral activation |
| `dbt_skills.md` | TIPP, ACCEPTS, DEAR MAN, PLEASE skills |
| `mindfulness_exercises.md` | Body scan, breath awareness, STOP technique |
| `grounding_techniques.md` | 5-4-3-2-1, box breathing, cold water |
| `sleep_hygiene.md` | CBT-I, sleep restriction, sleep efficiency formula |
| `crisis_resources.md` | 988, Crisis Text Line (741741), NAMI, SAMHSA |

## SSE Event Protocol

All events: `data: {JSON}\n\n`

| Type | Payload | Trigger |
|------|---------|---------|
| `token` | `content: str` | LLM token chunk (`on_chat_model_stream`) |
| `tool_use` | `tool: str` | Tool call starts (`on_tool_start`) |
| `tool_done` | `tool: str` | Tool call ends (`on_tool_end`) |
| `crisis` | `content: str` | Crisis keyword detected (pre-LLM) |
| `error` | `content: str` | Unhandled exception in stream |
| `done` | — | End of agent turn |

---

## Crisis Detection

Hard pre-LLM gate in `backend/crisis.py`. Runs before agent is invoked.

- `detect_crisis(message: str) -> bool` — case-insensitive substring match against keyword list
- Keywords include: `"want to die"`, `"kill myself"`, `"suicidal"`, `"self-harm"`, `"hurt myself"`, `"end my life"`, `"not worth living"`
- `CRISIS_RESPONSE` — static string: 988 Lifeline, Crisis Text Line (text HOME to 741741), NAMI, SAMHSA
- On trigger: stream `crisis` event + `done`, return immediately — LLM never called

---

## Session Management

```python
session_store: dict[str, list[BaseMessage]] = {}  # in-memory, lost on restart
```

- `session_id` is a UUID generated by `app.js` on page load
- Every turn: load history → append `HumanMessage` → pass full list to agent → append `AIMessage`
- Full history in every request = multi-turn context (follow-ups, pronoun refs, corrections)
- No TTL; persistence is post-MVP

## Startup Sequence

```bash
python backend/ingest.py          # once — builds faiss_index/
uvicorn backend.main:app --reload --port 8000
# open http://localhost:8000
```

If `faiss_index/` is missing, `GET /health` reports `"vector_store": "not_loaded"` and `rag_search` returns an error string rather than crashing.

**Environment variables:**

| Variable | Required | Default |
|----------|----------|---------|
| `OPENAI_API_KEY` | Yes | — |
| `TAVILY_API_KEY` | Yes | — |
| `PORT` | No | `8000` |
| `FAISS_INDEX_PATH` | No | `"faiss_index"` |
